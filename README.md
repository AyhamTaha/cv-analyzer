# CV Analyzer

**CS50 Final Project** by Ayham Abdulkareem Taha

---

## Project Description
CV Analyzer reads CVs and analyzes them to identify:
- Skills found
- Skills missing
- Personalized roadmap to acquire missing skills with estimated learning time and resources

This project helps users improve their CVs and learn new skills efficiently.

---

## Tech Stack
- Python 3.11
- Django 4.x
- Django REST Framework
- SQLite (default database)
- HTML/CSS/JS (if any frontend templates used)
